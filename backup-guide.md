# Guia de Backup

## Como fazer backup dos seus planos:

1. No sistema, use a função de exportação
2. Salve o arquivo .json em local seguro
3. Para restaurar, use a função de importação

## Dicas de segurança:
- Faça backup semanal
- Não compartilhe arquivos com dados sensíveis
- Use nomes genéricos para pacientes
